# install_workflow

My workflow installation.

It contains :
  - zsh config
  - vim config
  - nvim config
  - tmux config
  - other tools
  
## Build

Install on current user :
```
sudo ./install.sh
```

Install on another user :
```
sudo ./install.sh [USER]
```

If non superuser, use force option :

```
sudo ./install.sh -f
```
